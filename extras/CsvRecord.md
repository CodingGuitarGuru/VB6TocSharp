# CSV Record Class

## Example Usage

```c#
 public class PayComm : CsvRecord
    {
        [RecordField]
        public string Salesman = "";
        [RecordField]
        public string Lease = "";
        [RecordField]
        public string Name = "";
        [RecordField]
        public string Style = "";
        [RecordField]
        public string Landed = "";
   }
```
